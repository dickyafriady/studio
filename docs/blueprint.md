# **App Name**: Digital Canvas

## Core Features:

- Profile Showcase: Display personal information, a bio, and a profile picture, like on the reference site.
- Project Portfolio: Showcase personal projects with descriptions, images, and links.
- Contact Form: Enable visitors to send messages directly through the website.
- Blog Section: Publish articles or blog posts with a CMS.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) for calmness and trust.
- Background color: Very light grayish blue (#F0F8FF) providing a clean, unobtrusive backdrop.
- Accent color: Pale lavender (#E6E6FA) used sparingly for interactive elements.
- Body and headline font: 'Inter' for a modern, clean, and readable design.
- Use simple, line-based icons for navigation and to represent skills or interests.
- Maintain a clean, minimalist layout with ample whitespace to improve readability and focus.
- Subtle transitions and animations for a polished user experience. Animated page transitions, smooth scrolling, and interactive elements are all a must.
